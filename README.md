**CERTIGO – Barangay Indigency Certificate Request System
**
CERTIGO is a web-based application designed to automate and streamline the process of requesting Barangay Indigency Certificates in Barangay Balangkas, Valenzuela City. The system reduces reliance on manual procedures that often lead to delays, incomplete submissions, and inconvenience for residents. Through CERTIGO, citizens can submit requests online, upload requirements, and track the status of their applications in real time. Barangay personnel can efficiently review, approve, or decline requests through an organized admin dashboard.

Features

Online Request Submission – Residents can create an account, fill out the request form, and upload required documents.

Automated Request Number Generation – Ensures organized, traceable, and standardized request handling.

Real-Time Status Tracking – Users can monitor updates such as “Pending,” “For Verification,” “Approved,” or “Declined.”

Admin Dashboard – Barangay staff can manage applications, verify uploaded documents, update statuses, and generate records.

Secure Data Management – Built with privacy and accuracy in mind, aligning with the Data Privacy Act of 2012.

Technology Stack

Frontend: HTML, CSS, JavaScript

Backend: PHP

Database: MySQL

Architecture: Structured, modular design for maintainability and scalability

Purpose

CERTIGO enhances the efficiency, transparency, and reliability of local barangay services by reducing paperwork and processing time. It supports better data management, improves accountability, and offers residents a more convenient and accessible way to obtain important documents. The system contributes to more responsive and technology-driven local governance.
